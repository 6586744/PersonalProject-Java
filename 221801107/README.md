## wordCount

- 本项目是福州大学软工实践2021W班寒假第二次作业的仓库，只供学习交流。

## 开始

1. 下载相关依赖

```
npm install
```

或者

```
yarn
```

2. 修改 `package.json` 文件中的

```json
"start": "node src/index.js input.txt output.txt"
```

指定输出文件和输入文件，输入输出文件位置应与 `221801107/README.md` 文件同级

3. 开始运行

```
npm run start
```

或者

```
yarn start
```

4. 自动化单元测试

```
npm run test
```

或者

```
yarn test
```
